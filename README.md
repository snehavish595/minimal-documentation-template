# Minimal Documentation Template

A clean and lightweight HTML documentation template designed for creating project documentation, technical guides, API references, and internal manuals. Built with simplicity, readability, and ease of customization in mind.

##  Live Demo
 https://snehavish595.github.io/minimal-documentation-template/

---

##  Features

- Clean and minimal documentation layout  
- Sidebar navigation structure  
- Light and dark mode toggle  
- Code block styling  
- Responsive design  
- Easy to customize  
- No frameworks or build tools required  

---

## Folder Structure

```text
minimal-documentation-template/
├── index.html
├── documentation.html
├── README.md
├── README.txt
└── assets/
    ├── css/
    │   └── style.css
    └── js/
        └── main.js
```
##  Getting Started

This template works without any frameworks or build tools.

### Installation

1. Download the template files  
2. Unzip the package  
3. Open `index.html` in your browser  

---

##  Customization

- Update colors and layout styles using CSS variables defined at the top of `style.css`
- Edit navigation items directly in `index.html`
- Replace demo content with your own documentation text
